## 피드백 느낀점

### 잘한점
- 랜덤 부분 분리한 것 

### 개선점
- 강의를 보고나니 엄청 절차지향적으로 짰음
- 프로퍼티에 접근하기 보다 메시지를 보내는식으로 하기
- 간단한 평가문이여도 추후에 복잡해질 수 있으니 함수로 분리해보기
- 원시값 포장해서 유효성 검증하기
- 테스트코드 먼저 짜보기
- 귀찮아도 객체를 많이 만들기
